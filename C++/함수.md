# Stack Frame

이전에 배운 Stack Frame와 같이 Stack에는 매개 변수, 반환 주소값, 함수의 주소가 쌓이게 된다.

어셈블리 코드를 살펴 보면 알 수 있는데, 어셈블리 코드에서 함수가 끝나는 }부분에서 위의 Stack을 정리해준다.

예를들어 Add(int a, int b)를 호출 할경우

1. Stack에는 Add에 필요한 a,b를 넣고
2. Add를 호출한 위치를 호출한다.
3. 이후 지역변수를 쌓아서 Stack을 구상하게 된다.

- Cpp는 메모리를 직접 조작할 수 있기 때문에 문제가 생길 경우 어셈블리 언어를 통해 확인 할 수 있다.
